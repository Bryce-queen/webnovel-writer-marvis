# webnovel-writer 安装与使用说明

## 这是什么

长篇网文辅助创作系统。在 Marvis 中一键初始化小说、拆卷纲、写章节、审查质量、查询设定。

## 安装步骤

```
1. 将本文件夹放入 Marvis 的 skills/custom/webnovel-writer/

2. 安装依赖技能
   $ bash scripts/install-jwynia-deps.sh

3. 安装 Python 依赖
   $ python -m pip install -r scripts/requirements.txt

4. 重启 Marvis
```

> 第 2 步必须执行——它会安装 10 个 jwynia 创作技法技能，webnovel-writer 的 6 个核心管线（初始化/规划/写作/审查/学习/体检）都依赖它们进行诊断。

## 使用方法

重启 Marvis 后，直接对 Marvis 说以下任意触发词即可：

| 想做的事 | 对 Marvis 说 |
|---------|-------------|
| 创建新小说项目 | "帮我初始化一本都市异能类网文" |
| 规划某一卷的大纲 | "帮我规划第3卷的大纲" |
| 写一章新内容 | "帮我写第37章" |
| 审查章节质量 | "审查第37章的追读力和质量" |
| 查询角色/伏笔状态 | "我的主角现在能力达到什么水平了" |
| 保存好的写作经验 | "记住这个爽点写法" |
| 检查项目是否健康 | "体检一下我的项目" |
| 可视化项目概览 | "打开面板看看" |

## 依赖清单

以下是 webnovel-writer 调用的 10 个 jwynia 技法技能（由第 2 步自动安装）：

story-idea-generator、worldbuilding、genre-conventions、dialogue
scene-sequencing、plot-structure、story-sense、prose-style
prose-critique、revision

## 常见问题

**Q: 写章节时报"skill 不可用"？**
A: 说明 jwynia 技法技能未安装。执行 `bash scripts/install-jwynia-deps.sh` 后重启 Marvis。

**Q: RAG 检索功能需要配置什么？**
A: 在书项目根目录创建 `.env` 文件，配置 Embedding 和 Rerank API Key。不配置时自动退回 BM25 关键词检索。
